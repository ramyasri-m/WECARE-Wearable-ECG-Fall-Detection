# WECARE - Wearable ECG & Fall Detection


Project scaffold for ECG (MIT-BIH) and IMU (MobiAct/MobiFall) processing, modeling, and TFLite conversion.


Folders:
- data/: raw and processed datasets
- preprocessing/: preprocessing scripts for ECG & IMU
- features/: feature extraction
- models/: Keras model architectures
- train/: training scripts
- utils/: loaders and helpers
- experiments/: saved checkpoints and logs


See notebooks/ for quick experiments.
